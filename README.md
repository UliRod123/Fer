# Visual Assistant ✨

App de análisis creativo y mejora de imágenes con IA.

## Cómo subir a Streamlit Cloud (gratis)

### Paso 1 — Sube a GitHub
1. Ve a github.com y crea cuenta si no tienes
2. Clic en "New repository" → ponle nombre (ej: `visual-assistant`)
3. Marca "Private" si quieres que sea privado
4. Sube los dos archivos: `app.py` y `requirements.txt`

### Paso 2 — Conecta con Streamlit Cloud
1. Ve a share.streamlit.io
2. Inicia sesión con tu cuenta de GitHub
3. Clic en "New app"
4. Selecciona tu repositorio
5. En "Main file path" escribe: `app.py`
6. Clic en "Deploy"

### Paso 3 — Comparte el link
Streamlit te da un link tipo:
`https://tu-usuario-visual-assistant.streamlit.app`

¡Ese link es el que le mandas a ella! 🎁

## Funciones
- 🔆 Mejora automática inteligente de imagen
- 🔍 Upscaling HD ×2 con Lanczos
- 🧼 Remover fondo
- 📊 Análisis técnico (brillo, contraste, resolución)
- 🧠 Dirección creativa real con IA (LLaMA Vision vía Groq)
